Absolut path: /FONTS/src/drivers

Brief description

This directory contains all the scripts that allow testing the main functionalities implemented in the project.

Elements

  • DriverGame.java: Contains all the methods for executing and testing the main functionalities of the game.

    This driver has 3 main functionalities:

    • Create a new game: This method creates a new game with the specified parameters. The program will ask you for the size of the board, the number of regions, and the number of operands you want for your kenken.
    • Play a game: This method loads a game from a file. If a player already has a game in progress, the program will load this game and resume it. If not, the program will generate a random kenken
      • 1 | Get a hint
      • 2 | Game finished? Validate it!
      • 3 | Save game
      • 4 | Exit game without saving
      • | Press any other key to continue playing
    • Validate kenken: This method will automatically read a file with a kenken and validate it. If the kenken is correct, the program will print a message saying that the kenken is correct. If the kenken is incorrect, the program will print a message saying that the kenken is incorrect and will show the errors found.

  • DriverPlayer.java: Contains all the methods for executing and testing the main functionalities of the player.
