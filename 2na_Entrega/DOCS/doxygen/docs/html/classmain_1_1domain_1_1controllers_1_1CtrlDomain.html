Detailed Description

The CtrlDomain class serves as the main controller that coordinates interactions between various controllers (CtrlBoard, CtrlPlayer, CtrlGame, and CtrlPersistence) in the application. It provides methods to manage games, players, and the game's state, ensuring the functionality of starting, resuming, validating games, handling player operations, and persisting data.
