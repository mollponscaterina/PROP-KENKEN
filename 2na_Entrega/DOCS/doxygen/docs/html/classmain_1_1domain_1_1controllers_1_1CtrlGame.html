Detailed Description

The CtrlGame class serves as the controller for managing the state and actions of a KenKen game. It follows the singleton pattern to ensure only one instance of the controller exists. The class provides methods to validate the game, insert numbers into cells, resume games, and load players. It also offers methods to get hints and solutions for the game.
