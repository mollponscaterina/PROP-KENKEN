Detailed Description

The KenkenView class is responsible for managing the user interface for the KenKen puzzle game. It allows the user to interact with the puzzle, providing functionalities such as resetting the puzzle, leaving the game, getting hints, validating the current board, and viewing the solution. The KenkenView class provides an interactive interface for users to play KenKen puzzles, including options to validate their solutions, get hints, and reset or save their progress. The interface is visually styled with a dark blue background and golden accents, making it both functional and visually appealing.

IMPORTANT NOTE: For the next delivery, most of the attributes and methods will be stored in the Game and Board classes. We've been using them here for testing purposes. And knowing we don't have to deliver the code for this delivery, we wanted to develop faster. We will refactor the code in the next delivery.

◆ leavePopUp()

void main.presentation.views.KenkenView.leavePopUp ( )
inlineprivate

Displays a pop-up dialog asking the user if they want to leave the game. If the user confirms, another pop-up dialog asks if they want to save the game. If the user chooses to save the game, the game is saved using the CtrlPresentation class. If the user chooses not to save the game, the last game is removed using the CtrlPresentation class. Finally, the game view is displayed using the CtrlPresentation class and the current view is set to invisible.
